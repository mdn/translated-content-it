## Un esempio di base

Esploriamo come utilizzare il layout a più colonne, spesso chiamato _multicol_. Puoi seguire [scaricando il file di partenza multicol](https://github.com/mdn/learning-area/blob/main/css/css-layout/multicol/0-starting-point.html) e aggiungendo il CSS nei luoghi appropriati. Alla fine della sezione puoi vedere un esempio di come dovrebbe apparire il codice finale.

### Un layout a tre colonne

Il nostro file di partenza contiene HTML molto semplice: un wrapper con una classe di `container`, all'interno del quale si trova un'intestazione e alcuni paragrafi.

L'elemento {{htmlelement("div")}} con una classe di `container` diventerà il nostro contenitore multicol. Abilitiamo il multicol utilizzando una delle due proprietà: {{cssxref("column-count")}} o {{cssxref("column-width")}}. La proprietà `column-count` accetta un numero come valore e crea quel numero di colonne. Se aggiungi il seguente CSS al tuo foglio di stile e ricarichi la pagina, otterrai tre colonne:

```css
.container {
  column-count: 3;
}
```

Le colonne che crei hanno larghezze flessibili: il browser calcola quanto spazio assegnare a ciascuna colonna.

```css hidden
body {
  width: 90%;
  max-width: 900px;
  margin: 2em auto;
  font:
    0.9em/1.2 Arial,
    Helvetica,
    sans-serif;
}
```

### Impostare column-width

Modifica il tuo CSS per utilizzare `column-width` come segue:

```css
.container {
  column-width: 200px;
}
```

Ora il browser ti darà tante colonne quante può della dimensione che specifichi; lo spazio rimanente viene quindi condiviso tra le colonne esistenti. Questo significa che non otterrai esattamente la larghezza che specifichi a meno che il tuo contenitore non sia esattamente divisibile per quella larghezza.

```css hidden
body {
  width: 90%;
  max-width: 900px;
  margin: 2em auto;
  font:
    0.9em/1.2 Arial,
    Helvetica,
    sans-serif;
}
```

## Stilizzare le colonne

Le colonne create da multicol non possono essere stilizzate individualmente. Non c'è modo di rendere una colonna più grande delle altre o di cambiare il colore di sfondo o del testo di una singola colonna. Hai due opportunità per cambiare il modo in cui le colonne vengono visualizzate:

- Cambiare la dimensione dello spazio tra le colonne utilizzando la proprietà {{cssxref("column-gap")}}.
- Aggiungere una linea tra le colonne con la proprietà {{cssxref("column-rule")}}.

Usando il tuo esempio sopra, cambia la dimensione dello spazio aggiungendo una proprietà `column-gap`. Puoi sperimentare con valori differenti: la proprietà accetta qualsiasi unità di lunghezza.

Ora aggiungi una linea tra le colonne con `column-rule`. Allo stesso modo della proprietà {{cssxref("border")}} che hai incontrato nelle lezioni precedenti, `column-rule` è uno shorthand per {{cssxref("column-rule-color")}}, {{cssxref("column-rule-style")}}, e {{cssxref("column-rule-width")}}, e accetta gli stessi valori della proprietà `border`.

```css
.container {
  column-count: 3;
  column-gap: 20px;
  column-rule: 4px dotted rgb(79 185 227);
}
```

Prova ad aggiungere linee di stili e colori differenti.

```css hidden
body {
  width: 90%;
  max-width: 900px;
  margin: 2em auto;
  font:
    0.9em/1.2 Arial,
    Helvetica,
    sans-serif;
}
```

## Colonne che coprono

Puoi fare in modo che un elemento copra tutte le colonne. In questo caso, il contenuto si interrompe dove viene inserito l'elemento che copre e continua poi sotto l'elemento, creando un nuovo set di colonne. Per fare in modo che un elemento copra tutte le colonne, specifica il valore `all` per la proprietà {{cssxref("column-span")}}.

> [!NOTE]
> Non è possibile fare in modo che un elemento copra solo _alcune_ colonne. La proprietà può avere solo i valori `none` (che è il valore predefinito) o `all`.

```css hidden
body {
  width: 90%;
  max-width: 900px;
  margin: 2em auto;
  font:
    0.9em/1.2 Arial,
    Helvetica,
    sans-serif;
}
.container {
  column-count: 3;
  column-gap: 20px;
  column-rule: 4px dotted rgb(79 185 227);
}
h2 {
  column-span: all;
  background-color: rgb(79 185 227);
  color: white;
  padding: 0.5em;
}
```

## Colonne e frammentazione

Il contenuto di un layout a più colonne è frammentato. Essenzialmente si comporta nello stesso modo in cui si comporta il contenuto nei media paginati, come quando stampi una pagina web. Quando trasformi il tuo contenuto in un contenitore multicol, viene frammentato in colonne. Perché il contenuto possa fare questo, deve _rompersi_.

### Impostare break-inside

Per controllare questo comportamento, possiamo usare proprietà della specifica [CSS Fragmentation](/it/docs/Web/CSS/CSS_fragmentation). Questa specifica ci fornisce proprietà per controllare la rottura del contenuto in multicol e nei media paginati. Ad esempio, aggiungendo la proprietà {{cssxref("break-inside")}} con un valore di `avoid` alle regole per `.card`. Questo è il contenitore dell'intestazione e del testo, quindi non vogliamo che venga frammentato.

```css
.card {
  break-inside: avoid;
  background-color: rgb(207 232 220);
  border: 2px solid rgb(79 185 227);
  padding: 10px;
  margin: 0 0 1em 0;
}
```

L'aggiunta di questa proprietà fa sì che i box restino in un unico pezzo: ora non si _frammentano_ tra le colonne.
