Questa sezione elenca le domande relative agli strumenti/software che può usare per costruire siti web.

- [Quale software è necessario per costruire un sito web?](/it/docs/Learn_web_development/Howto/Tools_and_setup/What_software_do_I_need)
  - : In questo articolo spieghiamo quali componenti software sono necessari per modificare, caricare o visualizzare un sito web.
- [Quanto costa fare qualcosa sul web?](/it/docs/Learn_web_development/Howto/Tools_and_setup/How_much_does_it_cost)
  - : Quando lancia un sito web, può non spendere nulla o i suoi costi possono salire alle stelle. In questo articolo
    discutiamo quanto costa tutto e cosa ottiene per ciò che paga (o non paga).
- [Quali editor di testo sono disponibili?](/it/docs/Learn_web_development/Howto/Tools_and_setup/Available_text_editors)
  - : In questo articolo evidenziamo alcune considerazioni da fare quando si sceglie e installa un editor di testo per lo
    sviluppo web.
- [Cosa sono gli strumenti di sviluppo del browser?](/it/docs/Learn_web_development/Howto/Tools_and_setup/What_are_browser_developer_tools)
  - : Ogni browser dispone di un set di strumenti di sviluppo per il debug di HTML, CSS e altro codice web. Questo articolo spiega come
    utilizzare le funzioni di base degli strumenti di sviluppo del suo browser.
- [Come si assicura che il suo sito web funzioni correttamente?](/it/docs/Learn_web_development/Howto/Tools_and_setup/Checking_that_your_web_site_is_working_properly)
  - : Ha pubblicato il suo sito web online — molto bene! Ma è sicuro che funzioni correttamente? Questo articolo fornisce alcuni
    passaggi di risoluzione dei problemi di base.
- [Come si configura un server di test locale?](/it/docs/Learn_web_development/Howto/Tools_and_setup/set_up_a_local_testing_server)
  - : Questo articolo spiega come configurare un semplice server di test locale sul suo computer e le basi su come usarlo.
- [Come carica i file su un server web?](/it/docs/Learn_web_development/Howto/Tools_and_setup/Upload_files_to_a_web_server)
  - : Questo articolo mostra come pubblicare il suo sito online con strumenti {{Glossary("FTP", "FTP")}} — uno dei modi più comuni per mettere un sito web online in modo che altri possano accedervi dai loro computer.
- [Come uso GitHub Pages?](/it/docs/Learn_web_development/Howto/Tools_and_setup/Using_GitHub_pages)
  - : Questo articolo fornisce una guida di base alla pubblicazione di contenuti utilizzando la funzione gh-pages di GitHub.
- [Come ospitare il suo sito web su Google App Engine?](/it/docs/Learn_web_development/Howto/Tools_and_setup/How_do_you_host_your_website_on_Google_App_Engine)
  - : È alla ricerca di un posto per ospitare il suo sito web? Ecco una guida passo-passo per ospitare il suo sito web su Google App Engine.
- [Quali strumenti sono disponibili per il debugging e il miglioramento delle prestazioni del sito web?](https://firefox-source-docs.mozilla.org/devtools-user/performance/index.html)
  - : Questa serie di articoli le mostra come usare gli strumenti di sviluppo in Firefox per effettuare il debug e migliorare le prestazioni del suo
    sito web, utilizzando strumenti per controllare l'uso della memoria, l'albero delle chiamate JavaScript, il numero di nodi DOM resi, e
    altro ancora.
